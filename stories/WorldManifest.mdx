# World Manifest

The `WorldManifest` is the single configuration object passed to
[`generateWorld(manifest)`](https://github.com/lucas-stanford/dawnlike-atlas/blob/master/src/phaser/generators/world.js)
to produce one deterministic overworld. Spread
[`DEFAULT_WORLD_MANIFEST`](https://github.com/lucas-stanford/dawnlike-atlas/blob/master/src/phaser/generators/world.js)
into your own object to inherit defaults for the fields you don't set.

```js
import { generateWorld, DEFAULT_WORLD_MANIFEST }
  from 'dawnlike-atlas/roguelike/generators';

const world = generateWorld({
  ...DEFAULT_WORLD_MANIFEST,
  seed: 12345,
  width: 60,
  height: 40,
});
```

The live Wilderness example exposes the same knobs as Storybook controls — open
[Zone Examples / Wilderness](?path=/story/dawnlike-zone-examples-wilderness--wilderness)
to play with them visually.

## Returns

```ts
{
  width:    number,
  height:   number,
  tiles:    Array<Array<Tile>>,             // tiles[y][x]
  markers:  { townEntrance:{x,y}, dungeonEntrance:{x,y} },
  walkable: (x:number, y:number) => boolean,
  manifest: WorldManifest,                  // the normalized manifest
}
```

Each tile is
`{ type, tree, mountain, road, river, bridge, decor, marker }` — see
[`renderWorldTile`](https://github.com/lucas-stanford/dawnlike-atlas/blob/master/src/phaser/autotileRender.js)
for the rendering contract.
